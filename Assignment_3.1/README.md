State Statistics Management API

=> Build a complete REST API using Express.js that manages Indian state statistical data stored in an in-memory JSON array.

This API performs full REST operations including:

GET

POST

PUT

PATCH

DELETE

All data is stored and modified dynamically inside an in-memory array.
No database is used.


Sample API URLs
http://localhost:3000/states

http://localhost:3000/states/5

http://localhost:3000/states/highest-gdp

http://localhost:3000/states
